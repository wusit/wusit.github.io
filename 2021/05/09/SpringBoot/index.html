Spring / Spring MVC / SpringBoot

  • Spring是一个分层的轻量级开源框架，封装了一系列功能组件：SpringMVC / SpringAOP / Spring JDBC / Spring ORM / Spring Test等
  • SpringBoot是一个微服务框架，是Spring的扩展，消除了Spring所需的XML配置，使开发，测试和部署更加方便
  • Spring MVC < Spring < Spring Boot
  • 要深刻理解SpringBoot中各个注释的含义，比如@Service / @Component / @Resource
